function renderTrafego(metaAds,filtered,filteredEvents){ const norm=s=>String(s||'').toLowerCase().normalize('NFD').replace(/[\u0300-\u036f]/g,'').replace(/\s+/g,' ').trim(); // --- STEP 1: Monta spendMap da Meta por ad_name normalizado --- let spend=0; const spendMap={}; // chave: norm(ad_name) → {spend, adset, label} metaAds.forEach(ad=>{ spend+=parseFloat(ad.spend||0); const key=norm(ad.ad_name||''); if(!key)return; if(!spendMap[key])spendMap[key]={spend:0,adset:ad.adset_name||'',label:ad.ad_name||''}; spendMap[key].spend+=parseFloat(ad.spend||0); }); // --- STEP 2: Monta adMap agrupado pelo ad_name do CLOUDIA --- // Chave = norm(ad_name) do Cloudia const adMap={}; // Primeiro passa: registra todos os ad_names do Cloudia no período filtered.forEach(r=>{ const adName=String(r.ad_name||'').trim(); if(!adName)return; const key=norm(adName); if(!adMap[key]){ // Tenta casar com Meta para pegar spend e adset let metaInfo=spendMap[key]; if(!metaInfo){ // Busca parcial (primeiros 20 chars) const k20=key.substring(0,20); const found=Object.keys(spendMap).find(k=>k.startsWith(k20)||k20.startsWith(k.substring(0,20))); if(found)metaInfo=spendMap[found]; } adMap[key]={ spend: metaInfo?metaInfo.spend:0, adset: metaInfo?metaInfo.adset:'', label: adName, // usa o nome original do Cloudia leads:0, agend:0, comp:0, fechou:0, vendas:0, phones:new Set() }; } const ph=String(r.phone||'').replace(/\D/g,''); if(ph)adMap[key].phones.add(ph); }); // Atualiza leads = phones únicos Object.values(adMap).forEach(d=>{d.leads=d.phones.size;}); // --- STEP 3: Conta leads únicos totais --- const leadsPhones=new Set(); filtered.forEach(r=>{const ph=String(r.phone||'').replace(/\D/g,'');if(ph)leadsPhones.add(ph);}); const cloudiaLeadsCount=leadsPhones.size; // --- STEP 4: Distribui eventos por anúncio via telefone --- const allRows=[...filtered,...(filteredEvents||[])]; const seenFechou=new Set(); allRows.forEach(r=>{ const ph=String(r.phone||'').replace(/\D/g,''); if(!ph)return; // Encontra o adMap entry pelo telefone let matchKey=null; for(const [k,d] of Object.entries(adMap)){ if(d.phones.has(ph)){matchKey=k;break;} } if(!matchKey)return; if(isAgend(r.stage))adMap[matchKey].agend++; if(isComp(r.stage))adMap[matchKey].comp++; if(isFechou(r.stage)&&!seenFechou.has(ph+'_fechou')){ seenFechou.add(ph+'_fechou'); adMap[matchKey].fechou++; adMap[matchKey].vendas+=parseFloat(r.value||0); } }); // --- STEP 5: Leads sem ad_name --- const trackeadosPhones=new Set(); Object.values(adMap).forEach(d=>d.phones.forEach(p=>trackeadosPhones.add(p))); const semRastreioPhones=new Set([...leadsPhones].filter(p=>p&&!trackeadosPhones.has(p))); if(semRastreioPhones.size>0){ adMap['__sem_rastreio__']={spend:0,leads:semRastreioPhones.size,agend:0,comp:0,fechou:0,vendas:0,adset:'',label:'⚠ Sem rastreamento',phones:semRastreioPhones}; } // Origens const origenPhonesSeen={}; filtered.forEach(r=>{ const org=r.origin||'—'; const ph=String(r.phone||'').replace(/\D/g,''); if(!origenPhonesSeen[org])origenPhonesSeen[org]=new Set(); if(ph)origenPhonesSeen[org].add(ph); }); const origens={}; Object.keys(origenPhonesSeen).forEach(org=>{origens[org]=origenPhonesSeen[org].size;}); // Funil geral const evts=filteredEvents||filtered; const agendPhones=new Set(evts.filter(r=>isAgend(r.stage)).map(r=>String(r.phone||'').replace(/\D/g,''))); const agend=agendPhones.size; const compPhones=new Set(evts.filter(r=>isComp(r.stage)).map(r=>String(r.phone||'').replace(/\D/g,''))); const comp=compPhones.size; const fechados=evts.filter(r=>isFechou(r.stage)); const fechadosPhones=new Set(fechados.map(r=>String(r.phone||'').replace(/\D/g,''))); const contratos=fechadosPhones.size; const vendasMap={}; fechados.forEach(r=>{ const ph=String(r.phone||'').replace(/\D/g,'')||r.name; const v=parseFloat(r.value||0); if(!vendasMap[ph]||v>vendasMap[ph])vendasMap[ph]=v; }); const totalVendas=Object.values(vendasMap).reduce((s,v)=>s+v,0); const ticket=contratos>0?totalVendas/contratos:0; const leadsBase=cloudiaLeadsCount; const cplReal=leadsBase>0&&spend>0?spend/leadsBase:0; const cpa=agend>0?spend/agend:0; const cpcomp=comp>0?spend/comp:0; const cpc2=contratos>0?spend/contratos:0; document.getElementById('t-spend').textContent=fmt(spend); document.getElementById('t-cpl').textContent=cplReal>0?fmt(cplReal):'—'; document.getElementById('t-leads').innerHTML=fmtN(leadsBase)+' '; document.getElementById('t-leads-sub').textContent='leads únicos no período'; document.getElementById('t-agend').innerHTML=(agend||'0')+' '; document.getElementById('t-agend-sub').textContent=pct(agend,leadsBase)+' dos leads'; document.getElementById('t-comp').innerHTML=(comp||'0')+' '; document.getElementById('t-comp-sub').textContent=pct(comp,agend)+' show rate'; document.getElementById('t-contratos').innerHTML=(contratos||'0')+' '; document.getElementById('t-contratos-sub').textContent=pct(contratos,comp)+' de fechamento'; document.getElementById('t-conv').textContent=pct(contratos,leadsBase); document.getElementById('t-cpa').textContent=agend>0?fmt(cpa):'—'; document.getElementById('t-cpcomp').textContent=comp>0?fmt(cpcomp):'—'; document.getElementById('t-cpc2').textContent=contratos>0?fmt(cpc2):'—'; document.getElementById('t-oport').textContent=totalVendas>0?fmt(totalVendas):'—'; document.getElementById('t-vendas').textContent=totalVendas>0?fmt(totalVendas):'—'; document.getElementById('t-ticket').textContent=ticket>0?fmt(ticket):'—'; let totalEntradas=0; Object.values(_ccData||{}).forEach(d=>(d.orcamentos||[]).forEach(o=>totalEntradas+=Number(o.entrada)||0)); document.getElementById('t-entradas').textContent=totalEntradas>0?fmt(totalEntradas):'—'; const maxF=Math.max(leadsBase,agend,comp,contratos,1); document.getElementById('t-funnel').innerHTML=[ {label:'Leads',val:leadsBase,pct:'',color:'#A18B6C'}, {label:'Agendamentos',val:agend,pct:pct(agend,leadsBase),color:'#8a7358'}, {label:'Comparecimentos',val:comp,pct:pct(comp,agend),color:'#6b5940'}, {label:'Contratos',val:contratos,pct:pct(contratos,comp),color:'#4a3d2b'}, ].map(f=>`
${f.label}
${f.val||0}
${f.pct}
`).join(''); const totalO=Object.values(origens).reduce((a,b)=>a+b,0)||1; document.getElementById('t-origem').innerHTML=Object.entries(origens).sort((a,b)=>b[1]-a[1]).map(([o,n])=>`
${o}
${n}
${Math.round(n/totalO*100)}%
`).join('')||'Aguardando dados...'; window._lastAdMap=adMap; const adRows=Object.entries(adMap).sort((a,b)=>b[1].spend-a[1].spend||b[1].leads-a[1].leads); document.getElementById('t-ads-tbody').innerHTML=adRows.length>0?adRows.map(([key,d])=>{ const adCpl=d.leads>0&&d.spend>0?d.spend/d.leads:0; const adCpa=d.agend>0&&d.spend>0?d.spend/d.agend:0; const adCpcomp=d.comp>0&&d.spend>0?d.spend/d.comp:0; const adCpc=d.fechou>0&&d.spend>0?d.spend/d.fechou:0; const adTicket=d.fechou>0?d.vendas/d.fechou:0; return` ${d.label.substring(0,24)}${d.label.length>24?'...':''} ${(d.adset||'').substring(0,20)} ${d.spend>0?fmtK(d.spend):'—'} ${adCpl>0?fmt(adCpl):'—'} ${d.leads>0?d.leads+' ':'—'} ${d.agend>0?d.agend+' ':'—'} ${adCpa>0?fmt(adCpa):'—'} ${d.comp||'—'} ${adCpcomp>0?fmt(adCpcomp):'—'} ${d.fechou||'—'} ${adCpc>0?fmt(adCpc):'—'} ${d.vendas>0?fmtK(d.vendas):'—'} ${adTicket>0?fmt(adTicket):'—'} `; }).join(''):'Aguardando dados dos anúncios...'; }
